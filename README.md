# Sistema de consulta de estadísticas para atacantes de la Premier League


## Descripción
El fútbol es un deporte muy conocido a nivel mundial y una parte fundamental para analizar un partido son las estadísticas, con ellas se puede medir objetivamente el rendimiento de un jugador durante el encuentro. Es por lo anterior que se plantea un sistema que sirve para consultar estadísticas típicas de un atacante (Extremo izquierdo, Delantero centro ó extremo derecho), estas estadísticas son:
- Goles
- Asistencias
- Tiros
- Tiros a puerta
- Partidos jugados
- Minutos jugados
Lo anterior será analizado con 8 equipos de la misma liga para la temporada 2024-2025: 
- Manchester United
- Manchester City
- Liverpool
- Arsenal
- Chelsea
- Tottenham
- Aston Villa
- Newcastle

## Motivación
Aplicar los conocimientos de la clase de fundamentos de bases de datos para analizar una liga que a mi me gusta bastante.
